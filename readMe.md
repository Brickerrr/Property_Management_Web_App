# REManage Property Management System

REManage is a high-performance dashboard designed to bridge the gap between property data and actual management decisions. Developed by our group—Findlay, Mattus, Morgan, Callum, and Danny—this project provides a technical, streamlined interface for tracking everything from HMRC tax compliance to live on-site maintenance incidents.

### The Dashboard

The system is split into specialized views to handle the high density of information required by modern landlords:

* **Portfolio Index**: A searchable sidebar for quick access to every property and its current occupancy or compliance status.
* **Live Incidents**: A real-time feed that ranks reported issues by severity (Critical, Moderate, Low), complete with contractor dispatch tracking.
* **HMRC Oversight**: A dedicated panel for verifying Anti-Money Laundering (AML) status and Non-Resident Landlord (NRL) verification.
* **Compliance Audit**: Automates the tracking of Gas Safety, EICR, and EPC certificates with direct links to relevant UK legislation.

### Tech Stack

* **Backend/Templating**: EJS (Embedded JavaScript) for dynamic data injection.
* **Frontend**: Custom CSS using a Compliance Mode theme, featuring high-contrast UI elements like HMRC Blue and Contract Gold.
* **Logic**: Vanilla JS for instant filtering, search, and property switching without heavy framework overhead.
